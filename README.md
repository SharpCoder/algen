# algen

Algen is a platform, written in rust, for running genetic algorithms.

## Installation

Add the dependency to your crates

```toml
[package]
name = "my-app"
version = "0.1.0"
edition = "2021"

[dependencies]
algen = "0.1.0"
```

And then you can run cargo to fetch it.

```bash
cargo build
```
